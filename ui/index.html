OAuth was not the problem

Claim: “The OAuth implementation is broken — that's why the tests fail.” False. Jest never reached OAuth. Evidence: 

Error: ● Validation Error:

  Module <rootDir>/setEnvVars.js in the setupFiles option was not found.
         <rootDir> is: …/woocommerce-rest-api-ts-lib

  Configuration Documentation:
  https://jestjs.io/docs/configuration

Root cause: missing setEnvVars.js referenced by jest.config.ts (and gitignored). After restoring it, the hermetic nock suite passed without OAuth changes.
